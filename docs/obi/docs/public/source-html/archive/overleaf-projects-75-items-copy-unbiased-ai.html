Mathematical Derivations

For the marginal posterior computation:

\begin{align}
P(\theta|D) &= \int P(\theta,\phi|D)d\phi\\
&= \int \frac{P(D|\theta,\phi)P(\theta,\phi)}{P(D)}d\phi\\
&= \frac{1}{P(D)}\int P(D|\theta,\phi)P(\theta|\phi)P(\phi)d\phi
\end{align}
